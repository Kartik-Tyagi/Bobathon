# IBMer Shuffle - Quick Start Guide

Get the app running in 3 simple steps!

## 🚀 Quick Start

### Step 1: Install Dependencies
```bash
cd IBMerShuffle
npm install
```

### Step 2: Start the App
```bash
npm start
```

### Step 3: Open on Your Device
- **iOS**: Press `i` in terminal (requires macOS + Xcode)
- **Android**: Press `a` in terminal (requires Android Studio)
- **Phone**: Scan QR code with Expo Go app

That's it! The app should now be running.

## 📱 What You'll See

### Today Tab
- Daily IBMer with photo, bio, and achievements
- "Mark as Learned" button to track progress
- "Shuffle" button to explore random IBMers
- "Share Story" to share with colleagues

### Calendar Tab
- Full yearly calendar view
- Blue dots mark learned days
- Tap any date to see that day's IBMer
- Current day highlighted in IBM blue

### Streak Tab
- Current streak counter with flame icon
- Longest streak record
- Total IBMers learned
- Achievement badges (7, 30, 100 days)
- Progress bar to next milestone
- Confetti celebration button

### Library Tab
- Browse all 28 IBM legends
- Search by name, bio, or achievement
- View count and history tracking
- Tap any IBMer for full details

## 🔧 Troubleshooting

**TypeScript errors?**  
→ Normal before `npm install`. They'll disappear after installation.

**Metro bundler issues?**  
→ Run `npm start -- --clear` to clear cache

**Can't see the app?**  
→ Make sure Expo Go is installed on your phone

**Notifications not working?**  
→ Notifications only work on physical devices, not simulators
