QUICK INSTALLATION (3 STEPS)
-----------------------------

1. Open Terminal and navigate to the project:
   cd /Users/hassanhasan/Desktop/IBMerShuffle

2. Install all dependencies:
   npm install

3. Start the app:
   npm start

   Then press:
   - 'i' for iOS Simulator (macOS only)
   - 'a' for Android Emulator
   - Scan QR code with Expo Go app on your phone

TROUBLESHOOTING
---------------

Problem: "npm: command not found"
Solution: Install Node.js from https://nodejs.org/

Problem: TypeScript errors in editor
Solution: Normal before npm install. Run npm install first.

Problem: Metro bundler won't start
Solution: Run: npm start -- --clear

Problem: Can't connect to device
Solution: Make sure phone and computer are on same WiFi network
